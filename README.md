Este projeto visa demonstrar competência em engenharia de software, com foco no aprimoramento e complementação de habilidades em análise de dados.

As tarefas envolveram a criação e gerenciamento de ambientes virtuais Python, o desenvolvimento de um aplicativo web e sua implantação em um serviço de nuvem, garantindo sua acessibilidade ao público.

Neste projeto, utilizo um conjunto de dados de anúncios de vendas de automóveis.
